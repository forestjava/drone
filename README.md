# Battle Drone Simulator

Веб-приложение для симуляции управления боевым дроном с 3D визуализацией.

## Описание

Battle Drone Simulator отображает в веб-клиенте rigged 3D модель из ассета FBX-файла с элементами управления, используя Three.js.

## Структура проекта

### Ассеты
- **3D модели**: `public/models/` - FBX файлы дрона
- **Текстуры**: `public/textures/` - текстуры для 3D моделей
- **Звуки**: `public/sounds/` - аудио файлы

### Ветки
- **`main`** - сгенерированное LLM replit.com приложение
- **`client`** - чистое фронтенд React Vite приложение
- **`module`** - npm библиотека для разработки gamepad приложений (в разработке)

## Технологии

- **React 18** с TypeScript
- **Three.js** + **React Three Fiber** для 3D рендеринга
- **Vite** для сборки
- **Tailwind CSS** для стилизации
- **shadcn/ui** компоненты

## Установка и запуск

```bash
npm install
npm run dev
```
